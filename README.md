# eater-b-reporter

bouzuya's simple reporter for [eater][yosuke-furukawa/eater].

[yosuke-furukawa/eater]: https://github.com/yosuke-furukawa/eater

## Installation

```
$ npm install eater-b-reporter
```

## Usage

```js
$ eater --reporter eater-b-reporter
```
